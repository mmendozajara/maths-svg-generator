## Style Guide

Use a clean, professional visual style. Choose your own colours, fonts, and stroke widths, following these principles:

### Colours
- Use a dark colour (e.g. dark navy or black) for all text, strokes, labels, and outlines
- Use one accent colour for highlighted/emphasised elements (marked points, key measurements)
- Use a light pastel for filled/shaded regions (bar fills, pie slices, shape fills)
- Use a muted gray for secondary text (axis titles, captions)
- White background
- Light gray for grid lines

### Fonts
- Use a clean sans-serif font (Arial, Helvetica, or sans-serif) for all text
- Math variables (x, y, a, b) should be italic
- Font size: 14-16px for labels, minimum 11px for minor tick labels
- Bold for heading labels and vertex labels

### Strokes
- Shape outlines and axes: 2-3px, round linecap and linejoin
- Dimension/annotation lines and arrows: 1.5-2px
- Dashed lines for construction/hidden edges
- Grid lines: 0.5px light gray

### Arrows
- Graph axes: solid filled triangle arrowheads
- Dimension indicators: simple line arrowheads

## Diagram Type Guidelines

### Number Lines
- Horizontal axis with arrowheads at both ends
- Major ticks with number labels below, minor ticks without labels
- Marked points as filled circles above the line with bold labels
- Open circles for exclusive endpoints

### Coordinate Graphs
- Axes with arrowheads at positive ends, labelled
- Grid lines if precision needed. Tick marks with labels. Origin labelled
- Points as filled circles, labels offset. Curves as smooth polylines

### Geometric Shapes
- Outlines only by default (no fill unless shading explicitly requested)
- Vertex labels: capital letters, bold, positioned outside shape
- Side lengths at midpoint of edge, offset outward
- Right angle markers as small squares. Angle arcs with degree labels

### Bar Charts
- Vertical by default. Bars with fill and outline, evenly spaced with gaps
- Category labels centred below. Value labels on top of bars

### Pie Charts
- Circle centred with outline. Distinct fill for each slice
- Labels outside with leader lines. Small slices always labelled outside

### Fraction Diagrams
- Rectangles or circles divided into equal parts with clear borders
- Shaded parts filled, unshaded parts white. Label the fraction

## Quality Rules

1. **Precision**: Lengths, positions, and proportions must be mathematically accurate
2. **Readability**: All labels must be legible with adequate contrast
3. **Cleanliness**: No stray elements, no overlapping shapes or text
4. **Simplicity**: Do not add decorative elements unless requested. Clean and minimal
5. **Valid SVG**: Output must be well-formed XML that renders correctly in any browser
